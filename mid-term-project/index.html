Interactive Story Addendum

Development Process

During the development of our interactive story, we followed a structured process to create an engaging and immersive experience for players. This involved:

  • Planning the narrative flow and creating a detailed storyboard.
  • Implementing the HTML structure and styling the layout with CSS.
  • Writing JavaScript functions to handle the story progression and user choices.
  • Testing and refining the story to ensure all endings were reachable and images displayed correctly.

Challenges Faced

Throughout the development process, we encountered several challenges:

  • Ensuring the choices led to meaningful consequences and varied outcomes.
  • Implementing JavaScript functions to update the story text and images dynamically.
  • Testing the story to identify any bugs or inconsistencies in the branching narrative.

Overcoming Challenges

To overcome these challenges, we:

  • Collaborated closely to brainstorm creative solutions and refine the narrative structure.
  • Iterated on the JavaScript functions to ensure smooth transitions between story segments and choices.
  • Conducted thorough testing to identify and resolve any issues before finalizing the project.
